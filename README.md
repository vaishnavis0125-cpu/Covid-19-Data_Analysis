# Covid-19-Data_Analysis

## Project Overview
This project analyzes global COVID-19 data using Microsoft Ecel to uncover trends,patterns, and key insights relatd to confirmed cases, deaths, and recoveries across countries.

An interactive dashboard was developed to visualize important metrics and support data-driven decision making.
---

## Objectives
- Analyze country-wise COVID-19 statistics.
- Identify countries with the highest confirmed cases, deaths, and recoveries.
- Calculate important KPIs and ratios.
- Create an interactive dashboard for data exploration.
---

## Tool Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Cleaning Techniques
- Conditional Formatting
- GitHub
---

## Dataset
The dataset contains country-wise COVID-19 statistics including:

- Confirmed Cases
- New Cases
- Deaths
- New Deaths
- Recovered Cases
- Active Cases
- Serious/Critical Cases
- Cases per Million Population
- Deaths per Million Population

Source: Kaggle COVID-19 Dataset
---
## Data Cleaning Steps
- Removed duplicate records
- Handled missing values
- Standardized column names
- Corrected data types
- Performed validation checks

---

## Key Performance Indicators (KPIs)
- Total Confirmed Cases
- Total Deaths
- Total Recoveries
- Death Rate
- Recovery Rate
- Active Case Percentage

---

## Dashboard Features
- Country-wise comparison
- Top affected countries
- Recovery analysis
- Mortality analysis
- Interactive filters and slicers
- KPI cards
